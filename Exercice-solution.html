Description de l'entreprise

Tout d'abord, il est à noter que l'entreprise est Québecoise. Il s'agit d'une entreprise privée qui vise à aider les vendeurs et acheteurs de faire la transaction de ventes sans l'intermédiaires de courtiers et d'agences immobilières.

Besoin du grattage

En tant que futur propriétaire de maison, je serais intéressé à pouvoir récupérer les informations des deux sites webs afin de pouvoir faire une analyse des habitations qui ont un prix d'aubaine et pouvoir comparer le marché immobilier.

Je sais que le prix d'une maison est fortement influencé par le marché (prix) de son quartier, sa localisation, sa superficie habitable, sa superficie de terrain, le nombre de pièces et salles de bains.

Nous essairons d'extraires ces informations pour pouvoir consolider une liste des choix qui nous seraient les plus intéressants à nos critères factuellement.

Début de l'extraction avec Duproprio¶

Analyse des conditions d'utilisations et du "Robot.txt"

Il a été assez facile à retrouver : https://duproprio.com/robots.txt

Comme on le remarque ici, le site web permet à tous les agents mais ne permet pas de se promener avec un robot sur /browse/litings/all soit sans tous le listing des maisons.

En effet, lorsque l'on va à cette adresse, nous recevons un message

Nous serons donc prudent dans le choix de nos URL afin de se limiter à certaines bien spécifiques.

#Nous créerons une méthode pour récupérer précisément le contenu qui nous intéresse d'une page
def extrait_contenu_page_duproprio(contenu):
    #Parcourons le contenu de la page
    soup = BeautifulSoup(contenu, "html.parser")
    
    #On sait qu'il y a un nombre défini d'annonce à récupérer. Parcourons les
    annonces = []

    #On sait que les annonces sont dans les #li 
    for rows in soup.find_all("li"): 
        if(rows.has_attr('class') and "search-results-listings-list__item" in rows["class"] and "ad-zone" not in rows["class"]):
            #Nous sommes maintenant dans l'aperçu de l'annonce
            annonce = dict()
            
            #Sortons le numéro de l'annonce
            annonce['numero'] = rows.get('id')
                    
            #Sortons l'URL de l'annonce
            annonce['url'] = rows.find("a", class_="search-results-listings-list__item-image-link").get('href')
            
            #Sortons le montant demandé
            annonce['montant_demandé'] = rows.find("h2").text.strip()
            
            #Sortons l'adresse
            annonce['adresse'] = rows.find("div", class_="search-results-listings-list__item-description__item search-results-listings-list__item-description__address").text.strip()
            
            #Ajoute tous le contenu à annonce
            annonces.append(annonce)
            
            #pi_terrain = 0 #span #search-results-listings-list__item-description__characteristics--long-numbers
            #pi_habitable = 0 #span #search-results-listings-list__item-description__characteristics--long-numbers #
            #nb_chambre = 0 #div #search-results-listings-list__item-description__characteristics-popover #Chambres
            
            #print(rows)
    return annonces

#Créons une autre méthode générique pour parcourir les pages qui nous intéresses.
def extrait_nombre_defini_page_duprorio(nbPages):
    
    baseURL = "https://duproprio.com/fr/rechercher/liste?search=true&regions%5B0%5D=3&max_price=400000&rooms=3&is_for_sale=1&with_builders=1&parent=1&sort=-published_at&pageNumber="
    #On voit dans l'URL des éléments que nous pourrons altérer pour récupérer les données soit le numéro de la page

    annonces = []
    
    for page in range(nbPages):
        #Composons l'URL
        URL = baseURL + str(page)
        
        try:
            # Nous simulerons que nous sommes Chrome
            headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/79.0.3945.130 Safari/537.36'}

            #Récupérons le contenu de cette adresse
            response = requests.get(URL, headers=headers)
            
            #Extrayons le contenu et ajoutons le à un array
            annoncesRecues = []
            annoncesRecues = extrait_contenu_page_duproprio(response.text)
            
            annonces += annoncesRecues
            
            #On va donner à notre algorithme une pause tactique pour éviter de surcharger la source
            time.sleep(3)
            
        except requests.exceptions.RequestException as e:
            print(e)
            exit()
            
    return annonces
